# SOARy
A security orchestration, automation, and response (SOAR) tool created from grownd up from open source tools.


## Tech stack:
1. Elastic Stack (ELK)
2. Zeek (Bro)
3. Snort
4. MISP
5. TheHive
6. Node-RED
7. Python
8. Wazuh

## Key objective:
+ Dockerized SOAR to ensure flexibility of deployment & implementation
